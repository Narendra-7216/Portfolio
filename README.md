Portfolio site for Narendra — static HTML/CSS/JS

How to preview locally:

1. In a terminal run:

   python3 -m http.server 8000

2. Open http://localhost:8000 in your browser

What to customize:
- Replace assets/avatar.jpg and assets/Narendra_Resume.pdf with your photo and resume.
- Update projects in index.html with links to repositories and case studies.
- Add analytics or contact backend as needed.
